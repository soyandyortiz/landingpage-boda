# 💍 Landing Page - Invitación de Boda

Este proyecto es una **landing page de invitación de boda**, creada con **Next.js**, donde los invitados pueden ver toda la información importante del evento de forma moderna, elegante y accesible.

---

## 🚀 Cómo empezar

Para ejecutar el proyecto en tu computadora, primero abre la terminal y escribe:

```bash
npm run dev
